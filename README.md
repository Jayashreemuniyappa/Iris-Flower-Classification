# Iris Flower Classification

## Objective

This project aims to classify Iris flowers into three species: Setosa, Versicolor, and Virginica using Machine Learning algorithms.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Dataset

The Iris dataset is built into Scikit-learn and contains 150 flower samples with four features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The target variable consists of three flower species:

* Setosa
* Versicolor
* Virginica

## Exploratory Data Analysis (EDA)

* Dataset shape and structure
* Data type analysis
* Null value check
* Descriptive statistics
* Pairplot visualization
* Boxplot visualization

## Machine Learning Models

1. Logistic Regression
2. K-Nearest Neighbors (KNN)

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Classification Report

  * Precision
  * Recall
  * F1-Score

## Results

Both models were trained and evaluated on the Iris dataset. The model with the highest accuracy was selected as the best-performing model.

## Conclusion

The Iris flower species can be accurately classified using machine learning techniques. Petal Length and Petal Width were found to be the most important features for distinguishing between species.
